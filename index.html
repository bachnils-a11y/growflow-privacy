2. Welche Daten verarbeitet die App?

Grow Flow ist ein lokales Grow-/Pflanzen-Journal. Die App speichert die von Nutzerinnen und Nutzern eingegebenen Inhalte (z. B. Pflanzen, Notizen, Fotos, Pflege-Logs) ausschließlich auf dem Gerät.

3. Weitergabe an Dritte

Es erfolgt keine Weitergabe personenbezogener Daten an Dritte. Die App enthält keine Werbung, kein Tracking und keine Analyse-Dienste.

4. Internetverbindung / Server

Die App benötigt für die Kernfunktionen keine Serververbindung und erstellt keine Benutzerkonten.

5. Berechtigungen

Optional kann die App Zugriff auf folgende Funktionen anfragen:

  • Kamera (zum Erstellen von Pflanzenfotos)
  • Fotos (zum Auswählen/Speichern von Pflanzenfotos)
  • Mitteilungen (für lokale Erinnerungen, z. B. Pflege- oder Aufgaben-Reminder)

Berechtigungen können jederzeit in den iOS-Einstellungen geändert werden.

6. Speicherdauer

Die Daten bleiben auf dem Gerät gespeichert, bis sie in der App gelöscht oder die App deinstalliert wird.
